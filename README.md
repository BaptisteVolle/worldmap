# Worldmap
A interative world map with a list of all the countries generated by a geo-json file
The countries infomation are given by this API : https://restcountries.com/

## Use
- Click on a country yo have all its infromation displayed
- Click on one of the button to change the color country display ( Population, Area, Density or continent)
